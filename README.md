## Gin Gonic CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) application using Gin in Go.

### Prerequisites

Go 1.21.1 or later
MySQL or any other database supported by GORM

### Setup

```
git clone https://github.com/gin-gonic-crud.git
cd gin-gonic-crud
go mod tidy
go run main.go
```
